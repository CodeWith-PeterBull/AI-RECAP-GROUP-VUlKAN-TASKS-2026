# Practical Exercise 4 – Data Scrubbing

This repository contains the implementation of **Practical Exercise 4** for the Data Mining course.

The task focuses on **data understanding and data indexing** using the Netflix Titles dataset.

## Project Contents

- Netflix_Titles_Data_Cleaning.ipynb  
  Python notebook containing the analysis and answers to the knowledge check questions.

- netflix_titles.csv  
  Dataset used for the exercise.

- group_photo.jpg / meeting_minutes.pdf  
  Evidence of group collaboration.

## Technologies Used

- Python
- Pandas
